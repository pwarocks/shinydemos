General note

:target pseudo class used to create CSS-only tabbed navigation. <article> "tabs" are absolutely positioned on top of one another and given opacity: 0;, then transition is used to make tabs smoothly appear when the links that link to them are clicked, ie when they become a target (article[id*=tab]:target {}).
